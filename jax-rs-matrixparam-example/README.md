# JAX-RS @MatrixParam example

* https://www.favtuts.com/jax-rs-matrixparam-example/

# How to start

```bash
$ https://github.com/favtuts/java-jax-rs-tutorials.git

$ cd jax-rs-matrixparam-example
```

Run the web project on Tomcat or JBoss server, then test URLs:
* http://localhost:8080/jax-rs-matrixparam-example/books/2011/
* http://localhost:8080/jax-rs-matrixparam-example/books/2011;author=favtuts
* http://localhost:8080/jax-rs-matrixparam-example/books/2011;author=favtuts;country=malaysia
* http://localhost:8080/jax-rs-matrixparam-example/books/2011;country=malaysia;author=favtuts
